## Причины расторжения

| Код | Наименование для UI | Требуются документы |
|---|---|---:|
| `CLIENT_INITIATIVE` | По инициативе клиента | Нет |
| `DUPLICATE_CONTRACT` | Договор оформлен ошибочно или продублирован | Да |
| `PRODUCT_NOT_NEEDED` | Продукт больше не требуется | Нет |
| `PRICE_NOT_SUITABLE` | Не устраивает стоимость | Нет |
| `SERVICE_QUALITY` | Не устраивает качество обслуживания | Нет |
| `OTHER` | Другое | Нет |

## Каналы создания заявки

| Код REST / Client metadata | gRPC enum | Описание |
|---|---|---|
| `web` | `REQUEST_SOURCE_WEB` | Web-версия личного кабинета |
| `mobile` | `REQUEST_SOURCE_MOBILE` | Мобильное приложение |
| отсутствует / неизвестно | `REQUEST_SOURCE_UNSPECIFIED` | Канал не определен |

## Типы документов

| Код | Наименование | Форматы |
|---|---|---|
| `APPLICATION` | Заявление | PDF |
| `IDENTITY_DOCUMENT` | Документ, удостоверяющий личность | PDF, JPG, JPEG, PNG |
| `CONTRACT_COPY` | Копия договора | PDF, JPG, JPEG, PNG |
| `OTHER` | Прочий документ | PDF, JPG, JPEG, PNG |

## Статусы договора

| Код | Описание | Доступность расторжения |
|---|---|---:|
| `ACTIVE` | Договор активен | Да |
| `DRAFT` | Договор в черновике | Нет |
| `EXPIRED` | Срок действия договора истек | Нет |
| `TERMINATED` | Договор уже расторгнут | Нет |
| `CANCELLED` | Договор аннулирован | Нет |
| `SUSPENDED` | Договор приостановлен | Нет |
